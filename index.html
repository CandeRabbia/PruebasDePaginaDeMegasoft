Conocé nuestro sistema de gestión

Gestión para Empresas o Comercios

Sistema diseñado para la gestión administrativa y comercial de empresas. Incluye control fiscal e impositivo, módulos para puntos de venta y puestos administrativos, panel gerencial con acceso por usuario y contraseña, y niveles de permisos personalizados.

Más información

Gestión para Estación de Servicios

Sistema desarrollado para la administración integral de estaciones de servicio. Cubre aspectos operativos y fiscales, con módulos para playa y minimercado, panel central para operadores y herramientas de control para la gestión administrativa

Más información

Gestión para Supermercados

Sistema completo para la administración de supermercados. Incluye funciones fiscales y operativas en línea de cajas, panel para cajeros, módulo gerencial y herramientas para ventas, compras, stock, caja, bancos, contabilidad, estadísticas y cash flow.

Más información

Gestión para Cablevideo

Sistema integral para la administración de servicios de cable e internet. Permite parametrizar y liquidar servicios brindados. Incluye módulos de ventas, compras, caja, bancos, contabilidad, stock, servicios, recaudaciones, estadísticas y cash flow.

Más información

Gestión para Servicios Públicos

Sistema diseñado para la administración de cooperativas de servicios públicos. Permite parametrizar y liquidar servicios brindados. Incluye módulos de servicios, ventas, compras, caja, bancos, recaudaciones, contabilidad, stock y estadísticas.

Más información

Distribuidores OFICIALES Holistor

Representación y comercialización del software "Holistor Estudios Contables". Módulos integrados: impuestos, sueldos, contabilidad y registraciones. Multiusuario, multiempresa, sin licencias adicionales. Compatible con ARCA y con actualizaciones automáticas
